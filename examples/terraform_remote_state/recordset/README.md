# Yandex Cloud DNS Terraform module

Terraform module which creates Yandex Cloud DNS resources.

## Usage

To run this example you need to execute:

```bash
$ export YC_FOLDER_ID='folder_id'
$ terraform init
$ terraform plan
$ terraform apply
```
